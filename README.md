## 🛡️ Cybersecurity AI Chatbot (Streamlit + LangChain + Groq)


This project is an AI-powered cybersecurity chatbot built using Streamlit, LangChain, and Groq LLM. The application allows users to ask questions about common cyber attacks such as SQL Injection, Cross-Site Scripting (XSS), Brute Force attacks, and Phishing, and receive clear conceptual explanations along with defensive Python code examples.

The chatbot is designed for educational and lab environments, focusing on secure coding practices, detection mechanisms, and prevention techniques rather than generating exploit code.

## 🚀 Features

Interactive chat-based interface using Streamlit
Powered by LangChain and Groq Llama 3.1 model
Explains cybersecurity attacks conceptually
Provides defensive Python examples for detection and mitigation
Maintains chat history during the session
Simple and user-friendly AI learning tool for cybersecurity students

## 🛠️ Tech Stack

Python
Streamlit (Frontend UI)
LangChain (LLM orchestration)
Groq API (Llama-3.1-8B model)
dotenv for environment variable management

## 🎯 Use Cases

Learning cybersecurity fundamentals
Practicing secure coding techniques
Demonstrating AI-powered educational tools
Building LLM-based security assistants

## ⚠️ Disclaimer

This project is intended strictly for educational purposes. The chatbot focuses only on defensive techniques, attack detection, and mitigation strategies and does not generate exploit code.
